🗑️ Production Database Cleanup

⚠️ DANGEROUS OPERATION

⚠️ Warning: This Cannot Be Undone!

This operation will permanently delete ALL data from your production database except the admin account. This includes:

  • All users (except admin)
  • All jobs, quotes, and reviews
  • All payment records and invoices
  • All messages and support chats
  • All affiliate and partner data

📋 Before You Proceed

Make sure you:

  • Are logged into your admin account
  • Understand this will reset your database to a clean state
  • Have no real customers or data you want to keep
